📋 How It Works

This page uses the UMD bundle exactly like the original WebGazer.js: 

<script src="webgazer.js"></script>
<script>
  // window.webgazer is available ✓
  webgazer
    .setGazeListener((data, t) => {
      if (data) {
        console.log(data.x, data.y);
      }
    })
    .begin();
</script>

🔒 Camera & Privacy

What happens when you click "Start":

1. Your browser asks for webcam permission.
2. The video stream is processed entirely in your browser.
3. TensorFlow.js detects your face landmarks locally.
4. No video, images, or gaze data ever leave your device.
